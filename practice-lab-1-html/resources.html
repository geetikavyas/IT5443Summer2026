Suggested Practice Order

  1. Create the homepage named index.html.
  2. Add a header, navigation menu, main content area, and footer.
  3. Create a second page named resources.html.
  4. Add lists, a table, links, and images.
  5. Validate both pages using the W3C HTML Validator.
